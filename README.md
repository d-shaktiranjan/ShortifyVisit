## (A Flask Project)

# Setup

## Install Flask:
```bash
pip install flask
```

## Install Flask-SQLAlchemy:
```bash
pip install Flask-SQLAlchemy
```

## Create Database:
```bash
python
```
```bash
from main import db
```
```bash
db.create_all()
```

## Run Development Server

```bash
python main.py
```
## Public endpoint is at http://127.0.0.1:5000
